# reflection-self-love
An interactive self-reflection and mental wellness website built with HTML, CSS, and JavaScript featuring mood tracking, gratitude journaling, affirmations, relaxing ambient music, and a smart mirror experience.

Website ini mengajak pengguna untuk berhenti sejenak dari aktivitas sehari-hari dan meluangkan waktu untuk mengenali diri sendiri melalui afirmasi, jurnal rasa syukur, mood tracker, musik relaksasi, dan berbagai aktivitas sederhana lainnya.

## 📌 Main Features

### 🌸 Opening Experience
Animasi pembuka dengan narasi yang mengajak pengguna melakukan refleksi diri.

### 📷 Smart Mirror
Menggunakan kamera perangkat sebagai "cermin digital" untuk membantu pengguna melihat dirinya sambil membaca kalimat motivasi.

### 💖 Daily Affirmation
Menampilkan afirmasi positif secara acak untuk meningkatkan rasa percaya diri.

### 😊 Mood Tracker
Pengguna dapat memilih kondisi emosinya dan mendapatkan respon yang sesuai.

### 📖 Gratitude Journal
Media untuk menuliskan rasa syukur setiap hari.
Semua tulisan tersimpan menggunakan Local Storage sehingga tetap tersedia ketika website dibuka kembali.

### ✅ Daily Challenge
Checklist kebiasaan positif lengkap dengan progress bar.

### 🎵 Relaxing Music
Empat pilihan musik ambient:

- Rain
- Ocean
- Forest
- Piano

Musik diputar secara looping untuk membantu relaksasi.

### 🌙 Theme Switch
Pengguna dapat beralih antara Light Mode dan Dark Mode.

## Deskripsi Program

Reflection – Learn to Love Yourself merupakan sebuah website interaktif yang dikembangkan menggunakan HTML, CSS, dan JavaScript dengan tujuan membantu pengguna meningkatkan kesadaran diri (self-awareness), serta membangun kebiasaan positif dalam kehidupan sehari-hari.

Website ini menghadirkan berbagai fitur seperti Smart Mirror yang memanfaatkan kamera perangkat sebagai media refleksi diri, Daily Affirmation yang menampilkan kalimat-kalimat motivasi secara acak, Mood Tracker untuk mengenali kondisi emosional pengguna, Gratitude Journal sebagai tempat mencatat rasa syukur harian, Daily Challenge yang mendorong terbentuknya kebiasaan baik, serta Relaxing Music yang menyediakan musik ambient seperti suara hujan, hutan, laut, dan piano untuk menciptakan suasana yang lebih tenang.

Selain itu, aplikasi juga mendukung Light Mode dan Dark Mode, menyimpan data pengguna menggunakan Local Storage, serta memiliki antarmuka modern dengan konsep Glassmorphism yang responsif pada berbagai ukuran layar.

Website ini dikembangkan sebagai media sederhana yang mengajak pengguna meluangkan waktu beberapa menit setiap hari untuk lebih mengenal, menerima, dan menghargai dirinya sendiri.
